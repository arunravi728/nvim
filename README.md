# nvim
This is my basic NeoVim config. It was created based on [kickstart.nvim](https://github.com/dam9000/kickstart-modular.nvim) and [kickstart-modular](https://github.com/dam9000/kickstart-modular.nvim) with some major modifications to suite my specific workflow. The setup is meant to be minimal and highly portable with minimal dependencies some of which are - 
1. NeoVim Version 0.11+
2. Nerd Fonts
